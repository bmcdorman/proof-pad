Features

Define theorems...

(defun sum (xs)
   (if (endp xs)             ; If xs is empty
       0                     ; The sum is 0
       (+ (first xs)         ; Otherwise, add the first to... 
          (sum (rest xs))))) ; The rest of the items

...and prove them automatically
